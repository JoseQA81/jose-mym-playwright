# 🎭 jose-mym-playwright

Personal repository for practical Playwright learning and UI/E2E automation development.

## 🎯 Purpose

This repository is used to:

- learn Playwright from scratch and progressively move toward a more professional level
- practice selectors, locators, assertions, and test structure
- experiment with UI/E2E automation
- serve as a working base for **Phase 12 — Test Automation**
- test a minimal MCP Builder setup with Gemini CLI

## 🧰 Stack

- 🎭 Playwright
- 🔷 TypeScript
- 🥟 Bun
- 🤖 Gemini CLI
- 🧩 MCP Builder

## 🔌 Available MCPs

This repository is prepared to work with:

- Atlassian
- Context7
- Playwright
- Tavily

## ▶️ Run Playwright

```bash
npx playwright test

```

## ⚙️ Run MCP Builder

bun ai atlassian,context7,playwright,tavily

## 🔒 Notes

Sensitive files such as .env, .gemini/settings.json, and .gemini/settings.catalog.json are ignored and must not be committed.
