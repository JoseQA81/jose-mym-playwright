
# Playwright Locator Exercises



This folder preserves personal Playwright locator practice exercises originally created in the `refactor/login-page-pom` branch.



These exercises are kept as Markdown documentation on purpose:



- They are not part of the production E2E suite.

- They are not executed by Playwright CI.

- They are not compiled by TypeScript.

- They are intended as learning/reference material.



If any exercise becomes useful for the real automation suite, it should be reviewed, adapted, moved into `tests/`, validated locally, and submitted through a separate PR.
